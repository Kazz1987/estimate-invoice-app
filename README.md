# 見積〜請求管理Webアプリ

個人事業主向けの見積・請求業務一元管理アプリ。

## 技術スタック

| 区分 | 技術 |
|------|------|
| フロントエンド | React |
| バックエンド | Node.js (Express) |
| データベース | MySQL |
| PDF出力 | Puppeteer |
| デプロイ | Render |

## フェーズ1 機能

- 顧客管理
- 品目マスタ（大項目・小項目・単価）
- 見積書作成
- 請求書作成（見積から引き継ぎ）
- ステータス管理（見積→注文→納品→請求）
- PDF出力・印刷・印鑑欄

## 設計上の注意点

- 本アプリはシングルユーザー（個人事業主1名）によるローカル運用を想定した設計です
- 認証・認可機能は今回のスコープ外としています
- マルチユーザー対応が必要な場合はJWT認証の追加実装で対応可能です
- 現在のCORS設定はローカル開発用です。本番デプロイ時は環境変数`CORS_ORIGIN`で適切なオリジンを指定してください
- 本アプリはPC利用を想定したUI設計です。見積・請求書管理という業務特性上、デスクトップでの利用を主眼に置いています。スマートフォン・タブレット向けのレスポンシブ対応は今回のスコープ外としています。
